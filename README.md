# 📝 MERN Blog Platform

A full-stack blogging platform built with the **MERN Stack** where users can create, manage, edit, and share blogs with image uploads, authentication, likes, comments, search, and a personal dashboard.

> Built as a production-ready portfolio project using modern React and Node.js best practices.

---

- Persistent Login
